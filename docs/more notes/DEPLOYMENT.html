IIS deployment checklist

  1. Publish Bupa.Ipac.Web with Windows Authentication enabled
  2. Run correct npm run build-* before publish
  3. Configure Web.config connection strings and CRM/WCF endpoints for target environment
  4. Install Windows Service on app server(s)
  5. Verify AD department authorization and CRM connectivity
