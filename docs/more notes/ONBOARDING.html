Exercise 1: Trace a save (2 hours)

  1. Set breakpoint in OpportunityService.saveOpportunity (Angular)
  2. Click Save in UI
  3. Follow POST in Network tab
  4. Break in OpportunityController.SaveOpportunity
  5. Step through OpportunityService.SaveOpportunity (.NET)
  6. Identify 3 stored procedures called

Exercise 2: Trace authentication (1 hour)

  1. Break on GetAuthenticatedUser
  2. Inspect AD department string
  3. Step into SetPrincipalCRM
  4. Inspect SessionManager.IPUser in watch window

Exercise 3: Add a read-only API call (half day)

  1. Add method to appropriate I*Service + implementation
  2. Add repository method calling existing SP or new SP
  3. Add controller action with [Authorize] + [SessionExpireFilter]
  4. Add session.POST wrapper and call from component
